MediaWiki export in Etherpad
============================

Use the normal editbar buttons to add MediaWiki style markdown.
Export as MediaWiki.

To enable MediaWiki view click Settings -> MediaWiki

Setting as default
==================
Paste the below into your settings.

"ep_MediaWiki_default": true,
