# ep_real_time_chat
Etherpad plugin to send real time chat updates

## Configuring

Add the below settings to settings.json or throught the settings UI
```
  "ep_real_time_chat":{
    "realTimeChatOnByDefault":true,
    "forceRealTimeChat": true
  }
```

It should be obvious what they do..
